# 一个库 os
## 使用方式
    ```
    文件目录操作
        os.getcwd()     #获取路径
        os.listdir()    #获取目录下的文件
        os.makedirs()   #新建一个目录在本目录下
        os.path.exists()    #该路径是否存在

    文件管理系统
        os.removedirs() 
        shutil.rmtree()
        os.rename()

    文件目录多种检验
        os.path.isfile()
        os.path.exists()
        os.path.isdir()
        os.path.basename()
        os.path.dirname()
        os.path.split()
    ```
